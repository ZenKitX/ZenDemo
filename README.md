# ZenDemo

Zen系列Flutter模板库

## 项目结构

```
lib/
  ├── main.dart           # 应用入口
  ├── core/              # 核心功能
  ├── features/          # 功能模块
  ├── shared/            # 共享组件
  └── utils/             # 工具类
```

## 开始使用

### 安装依赖

```bash
flutter pub get
```

### 运行项目

```bash
flutter run
```

## 开发规范

- 使用 Material 3 设计规范
- 遵循 Flutter 官方代码风格
- 组件化开发，模块解耦
